== Description ==
Плагин интегрирует бонусную систему в ваш WooCommerce магазин с использованием API от сервиса "Бонус Плюс"

Возможности плагина:
* Добавление постоянной скидки на сайт
* Возможность списания балов на странице оформления заказа
* Автоматическая регистрация в бонусной системе при регистрации на сайте или при оплате заказа
* Отдельная страница в личном кабинете для получения информации о скидках и баллах
* Настраиваемые всплывающие сообщения с рекламой бонусной системы
* Каждую возможность можно включить отдельно от остальных

[Официальный сайт Бонус Плюс](https://bonusplus.pro)

[Обработка персональных данных](https://bonusplus.pro/new/data-processing)

[Документация для разработчика (API)](https://bonusplus.pro/api/Help)
